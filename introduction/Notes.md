# Class Notes (May 15)

While_

Um ciclo while é sempre executado enquanto a condição especificada não seja availada como verdadeira.

*Sintax:

while (condição) {
    declaração
}

*Exemplo:

let j = 0;
while (j <= 6) {
    console.log(j);
    j++;
}

No exemplo acima, o codigo é repetido até chegar ao 6 (porque o j não ia é "true" se for mais que 6)
